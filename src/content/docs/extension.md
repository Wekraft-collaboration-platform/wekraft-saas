# VS Code Extension

The **Wekraft VS Code Extension** is the heart of the developer experience. It allows you to stay in your flow state while staying connected to your team.

## Features
- **Task Management**: View, start, and complete tasks without leaving your editor.
- **Deep Links**: Open specific issues or tasks directly from the web app into VS Code.
- **Auto Time-Tracking**: Automatically log your development time to specific tasks.
- **Seamless Auth**: One-click authentication using our secure handshake protocol.

## Installation
1. Search for "Wekraft" in the VS Code Marketplace.
2. Click **Install**.
3. Open the Wekraft tab in the sidebar and click **Login**.
4. You will be redirected to the web app to authorize the connection.

## Why use the extension?
Developers spend 90% of their time in the IDE. By bringing project management into the IDE, we eliminate the friction of context switching.
