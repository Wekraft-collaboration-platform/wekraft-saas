# Tasks

Tasks are the fundamental units of work in Wekraft. They help you break down complex projects into manageable steps.

## Task Properties
- **Priority**: High, Medium, or Low.
- **Status**: Todo, In Progress, Review, or Done.
- **Assignee**: The team member responsible for the task.
- **Due Date**: When the task needs to be completed.

## Working with Tasks
You can view tasks in multiple layouts:
- **List View**: For a dense, information-rich overview.
- **Board View**: A Kanban-style interface for tracking flow.
- **Calendar View**: To see how tasks are distributed over time.

### Smart Sync
Tasks updated in the VS Code extension reflect instantly on the web dashboard.
