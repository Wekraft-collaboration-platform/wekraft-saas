# Projects

Projects are the primary containers for all your work in Wekraft. Each project brings together tasks, issues, and team members.

## Creating a Project
1. Navigate to the **Projects** tab in the main dashboard.
2. Click **New Project**.
3. Enter a name, description, and select a color theme.
4. Invite your team members via email.

## Project Dashboard
The project dashboard provides a high-level view of:
- **Project Progress**: Percentage of completed tasks.
- **Member Activity**: Who is currently active.
- **Upcoming Deadlines**: Sprints or tasks ending soon.

## Integration
Each project can be linked to a GitHub repository, allowing for automatic syncing of commits and pull requests.
